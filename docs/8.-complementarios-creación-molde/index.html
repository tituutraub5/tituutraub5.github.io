8. Complementarios 2- Creación de Moldes

Modelado moldes

Es el proceso de fabricación donde a un molde de cilicona se le vierte recina líquida compuesta por líquidos que permiten la solidificación.

Modelo Final

Pasos a realizar

Primero vamos a ir al módulo de superficies y hacer un sketch nuevo en el plano front. Vamos a ir a línea para hacer un eje de simetría

Y hacemos la siguiente geometría

Después vamos a ir al comando revolve para que se genere un objeto a partir de la geometría seleccionada.

Después vamos a hacer un pequeño fillet para aumentar la curvatura de las esquinas.

Entonces lo que queremos hacer ahora es que nuestro objeto sea la geometría base para hacer nuestro molde. Para eso vamos a ir a sketch, elegir la cara inferior, crear un rectángulo desde el centro, tomar el punto central y hacer un cuadrado que sobrepase la forma de la cara inferior.

Vamos a terminar el sketch y extruir el rectángulo con el propósito de sobrepasar el objeto original.

Después, para distinguir nuestro objeto original le aplicaremos un color brillante amarillo apretanndo la letra A de nuetsra tecla

Después vamos a ir a la función de combinar y vamos a seleccionar como el target la caja y el tool bodie nuestro objeto original. Usaremos las siguientes indicaciones.

Y así nos queda nuestro molde

Después, vamos a inspeccionar el molde para editar aquellas partes que son rectas y difíciles de desmoldar.

Despúes vamos a hacer un nuevo sketch en el plano frontal, y en crear proyecto (intersectar la geometría),seleccionando nuestro cuerpo original.

Después vamos a hacer una línea de referencia y vamos a seleccionar todas las líneas y dejarlas como líneas de construcción.

Después hacemos otra línea y la interceptamos donde queremos hacer el ángulo de desmolde

Ahora vamos a hacer un revolve de nuestra nueva geometría

Y finalmente quedó el ángulo de desmolde para poder sacar la pieza cuando necesitemos retirarla del molde.
