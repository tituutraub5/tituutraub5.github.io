6. Impresión 3D CURA

Utilización software impresión 3D Ultimaker Cura

1. Introducción

Este software pertenece a la categoría de los slicer. Ultimaker Cura es el software de impresión 3D más popular del mundo. La impresión se prepara con unos cuantos clics, se integra con el software de CAD para un mejor flujo de trabajo y se puede aprovechar las configuraciones personalizadas para conseguir un control más completo.

Lo que primero vamos a hacer es setear nuestra impresora apretando nuestro menú y seleccionaremos la “Creality Ender-3” que está en la sección de impresoras no conectadas. Los presets los dejamos como venían preestablecidos.

Segundo, vamos a ir al material y cabezal. En el material vamos a escoger el purple PLA+ y el cabezal (Nozzle) de 0.4 mm

Después, en fusion 360 vamos a exportar nuestra pieza que queremos imprimir en STL para importarla en CURA. Para eso vamos a herramientas y crear 3d print.

Y vamos a dejar el siguiente formato y aceptar para guardar

Después vamos a abrir el objeto STL en Cura

Después, en print settings vamos a cambiar el layer height a 0.32 mm. Después vamos a apretar slice para que se procese la pieza.

Si no nos gusta el proceso del modelo, podemos cambiar algunos parámetros en la misma tabla de print settings. Todo esto se puede observar en el preview.
