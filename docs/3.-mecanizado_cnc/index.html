3. Mecanizado cnc

Encargo Mecanizado CNC para una silla y asiento

1. Introducción

¿Qué es el mecanizado CNC?

El mecanizado CNC son máquinas-herramientas automatizadas que se controlan por el computador. Mediante herramientas rotativas de múltiples filos de corte y con poquísima intervención humana, pueden llevar a cabo diferentes operaciones sobre una pieza. Este tipo de mecanizado se ha aplicado en múltiples operaciones de desbaste o de arranque de viruta. Pero también torneado, fresado, taladrado, incluso complementando todas ellas en una sola máquina.

Existen dos tipos de máquinas CNC: verticales y horizontales. La diferencia radica en la posición del motor del husillo. El tamaño también dependerá según el tipo de pieza a fabricar. Puede ser de 3, 4 y 5 ejes. A mayor número de ejes, mayor rapidez de producción de piezas más complejas.

Todas las máquinas herramientas pueden ser adaptadas a un proceso de fabricación mediante comandos CNC. El mantener un control y una lectura constante de los estados operacionales, lo hace un proceso de fabricación seguro. Tiene además mucha más precisión y calidad en los diversos acabados superficiales obtenidos.

2. Objetivo del trabajo

Este trabajo trata de cortar piezas en una plancha de 122cm x 122cm x 15 mm para poder formar una silla. Esta silla tiene diversas partes en su anatomía, las cuales son:

  • Asiento
  • 2 Largueros
  • Patas Frontales A y B
  • Patas Posteriores A y B
  • 2 Travesaños
  • Respaldo

Silla

Asiento

Estas partes las dividiremos en 2 debido a que la curvatura del asiento tiene otro formato de trabajo que vamos a especificar más adelante.

2. Objetos

Para poder cortar la plancha de 122cm x 122cm x 15 mm necesitaremos una fresa con las siguientes especificaciones: Autodesk Generic 3-Axis (#1 – 6 mm diámetro Flat Endmill).

Esta fresa debe pasar múltiples veces en la plancha para poder hacer el corte, y es óptimo seleccionar un corte de 3 mm por pasada.

3 Proceso de corte de la silla

Como vemos en el video, el corte de las partes de la silla comienza con un 2D Contour para poder hacer los agujeros que después harán el nexo con las otras partes. Siguiente, se hace un 2D Pocket para poder hacer los huecos de encaje entre las patas frontales y posteriores. Por último, se hace de nuevo un 2D Contour para poder hacer el corte final en las piezas, haciendo stocks de 3 mm entremedio para no romper las piezas.

4 Proceso de corte Asiento

Como vemos en el video, primero se parte con un Contour 2D para poder hacer el corte exterior en las piezas, haciendo stocks de 3 mm entremedio para no romper la plancha. Después se hace un Trace para poder hacer la curvatura del asiento.
